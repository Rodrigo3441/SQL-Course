## Functions

You will use functions every time you want to do these things with data:

- Manipulate
- Analyze
- Transform
- Clean

Functions are a built-in SQL Code that:

- Accepts an input value
- Processes it
- Returns an output value

## Two Functions Categories

- **Single-row functions**: receives one value and return only one value
- **Multi-row functions:** receives many values and returns one or more values

## Nested Functions

Function used inside another function

## Types of SQL Functions

### Single-row Functions

- String Functions
- Numeric Functions
- Date and Time Functions
- NULL Functions
### Multi-row Functions

- Aggregate Functions (basics)
- Window Functions (advanced)
