## ROUND

Approximates a number to the number of decimal points chosen  
example:

3.516 and ROUND 2 = 3.52
3.516 and ROUND 1 = 3.5
3.516 and ROUND 0 = 4

## CEIL / CEILING

Round a number up to the nearest integer, regardless of wheter the decimal part is greater than or less than 0.5

| Example              | Output |
| -------------------- | ------ |
| SELECT (CEIL(12.34)) | 13     |

## FLOOR

Round a number down to the nearest integer, ignoring the decimal part

| Example             | Output |
| ------------------- | ------ |
| SELECT FLOOR(12.98) | 12     |
