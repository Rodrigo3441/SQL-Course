# SQL Course Notes

My personal notes from the **SQL Classes – From Zero into Advanced** course by Data with Baraa.

This repository contains lecture summaries, examples, screenshots, and concepts covered throughout the course.

> These notes were created while studying SQL and are intended for educational purposes and future reference.

## Repository Structure

```text
SQL-Course/
├── lectures/
├── images/
└── README.md
```

- `lectures/` → Markdown notes for each lecture.
- `images/` → Images and screenshots used in the notes.

## Course Topics

Some of the topics covered include:

- SQL Fundamentals
- SELECT Statements
- DDL (CREATE, ALTER, DROP)
- DML (INSERT, UPDATE, DELETE)
- WHERE Conditions
- JOINS
- Set Operators
- String and Number Functions
- Date and Time Functions
- NULL Handling
